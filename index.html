Three AI powerhouses.
One container. Drei KI-Schwergewichte.
Ein Container. Üç yapay zeka devi.
Tek container.

Each tool runs in bypass mode by default for fully autonomous operation. Need the standard approval flow? Just append -normal to any command. Jedes Tool läuft standardmäßig im Bypass-Modus für vollständig autonomes Arbeiten. Du brauchst den normalen Freigabeprozess? Hänge einfach -normal an jeden Befehl an. Her araç varsayılan olarak bypass modunda çalışır. Standart onay akışı gerekiyorsa komutun sonuna -normal ekleyin.

C

Claude

Anthropic's Claude CLI with full permissions. Reads, writes, and executes without asking. The backbone of autonomous coding sessions.

Anthropics Claude CLI mit vollen Berechtigungen. Liest, schreibt und führt aus, ohne nachzufragen. Das Rückgrat autonomer Coding-Sessions.

Anthropic'in Claude CLI'sı, tam yetkilerle. Sormadan okur, yazar ve çalıştırır. Otonom kodlama oturumlarının temel taşı.

claude --dangerously-skip-permissions claude-normal
Cx

Codex

OpenAI's Codex CLI with full sandbox bypass. Approvals and sandboxing disabled so it can operate at full speed.

OpenAIs Codex CLI mit vollem Sandbox-Bypass. Freigaben und Sandboxing sind deaktiviert, damit es mit voller Geschwindigkeit arbeiten kann.

OpenAI'ın Codex CLI'sı, tam sandbox bypass ile. Onaylar ve sandbox devre dışı bırakıldı, tam hızda çalışır.

codex --dangerously-bypass-approvals-and-sandbox codex-normal
G

Gemini

Google's Gemini CLI in yolo mode. No confirmations, no guardrails. Just direct execution of whatever needs to happen.

Googles Gemini CLI im Yolo-Modus. Keine Bestätigungen, keine Leitplanken. Einfach direkte Ausführung von allem, was nötig ist.

Google'ın Gemini CLI'sı yolo modunda. Onay yok, korkuluk yok. Ne gerekiyorsa doğrudan çalıştırır.

gemini --yolo gemini-normal
Features Funktionen Özellikler

Built for unattended work. Gebaut für unbeaufsichtigtes Arbeiten. Gözetimsiz çalışma için tasarlandı.

⚡

One Script Setup

Ein-Script-Setup

Tek Script Kurulum

Run one bash script on a fresh Linux machine. It installs Docker, generates all config files, builds the image. Nothing to clone, nothing to configure manually.

Führe ein einziges Bash-Script auf einer frischen Linux-Maschine aus. Es installiert Docker, generiert alle Konfigurationsdateien und baut das Image. Nichts zu klonen, nichts manuell zu konfigurieren.

Temiz bir Linux makinesinde tek bir bash scripti çalıştırın. Docker'ı kurar, tüm konfigürasyon dosyalarını oluşturur, imajı derler. Klonlama yok, manuel ayar yok.

📦

Complete Sandbox

Vollständige Sandbox

Tam Sandbox

Everything runs inside Docker. Your host machine stays untouched. AI agents get root inside the container, not on your system.

Alles läuft innerhalb von Docker. Dein Host-System bleibt unberührt. KI-Agenten bekommen Root im Container, nicht auf deinem System.

Her şey Docker içinde çalışır. Ana makineniz dokunulmaz kalır. Yapay zeka ajanları container içinde root alır, sisteminizde değil.

🐳

Docker-in-Docker

The container has access to the host Docker socket. AI agents can build images, spin up services, run tests in containers, and manage full deployment pipelines.

Der Container hat Zugriff auf den Docker-Socket des Hosts. KI-Agenten können Images bauen, Services starten, Tests in Containern ausführen und komplette Deployment-Pipelines verwalten.

Container, host Docker soketine erişebilir. Yapay zeka ajanları imaj oluşturabilir, servis başlatabilir, container içinde test çalıştırabilir ve tam deployment pipeline'ları yönetebilir.

🔄

Self-Healing Wrappers

Selbstheilende Wrapper

Kendini Onaran Wrapper'lar

A cron-based watchdog checks every 5 minutes. If a CLI auto-updates and overwrites its wrapper, the watchdog restores bypass mode automatically.

Ein Cron-basierter Watchdog prüft alle 5 Minuten. Wenn ein CLI sich automatisch aktualisiert und seinen Wrapper überschreibt, stellt der Watchdog den Bypass-Modus automatisch wieder her.

Cron tabanlı bir watchdog her 5 dakikada kontrol eder. Bir CLI otomatik güncellenip wrapper'ını üzerine yazarsa, watchdog bypass modunu otomatik olarak geri yükler.

💻

Persistent Sessions

Persistente Sitzungen

Kalıcı Oturumlar

Run ./enter.sh to drop into the container and start working. Use tmux inside to keep AI agents running after you disconnect. Come back anytime and pick up where you left off.

Starte ./enter.sh, um den Container zu betreten und loszulegen. Nutze tmux darin, um KI-Agenten weiterlaufen zu lassen, nachdem du dich abgemeldet hast. Komm jederzeit zurück und mach weiter, wo du aufgehört hast.

./enter.sh ile container'a girin ve çalışmaya başlayın. Bağlantıyı kestikten sonra yapay zeka ajanlarının çalışmaya devam etmesi için içeride tmux kullanın. Dilediğiniz zaman geri dönüp kaldığınız yerden devam edin.

🛡

No Credentials in Code

Keine Zugangsdaten im Code

Kodda Kimlik Bilgisi Yok

The setup script contains zero API keys, tokens, or secrets. You authenticate each CLI interactively inside the container. Credentials stay in the container's home directory, never on your host.

Das Setup-Script enthält keine API-Schlüssel, Tokens oder Secrets. Du authentifizierst jedes CLI interaktiv im Container. Zugangsdaten bleiben im Home-Verzeichnis des Containers, nie auf deinem Host.

Kurulum scripti sıfır API anahtarı, token veya gizli bilgi içerir. Her CLI'yı container içinde interaktif olarak kimlik doğrularsınız. Kimlik bilgileri container'ın ev dizininde kalır, asla ana makinenizde değil.

🌐

Works with Claude Remote

Funktioniert mit Claude Remote

Claude Remote ile Çalışır

Pair with Claude's SSH remote execution for the best experience. Since everything is already sandboxed in Docker, you need far fewer permission approvals than running on bare metal. The container is the trust boundary.

Kombiniere es mit Claudes SSH-Remote-Ausführung für das beste Erlebnis. Da alles bereits in Docker sandboxed ist, brauchst du weit weniger Berechtigungsgenehmigungen als auf Bare Metal. Der Container ist die Vertrauensgrenze.

En iyi deneyim için Claude'un SSH uzaktan çalıştırma özelliğiyle birleştirin. Her şey zaten Docker içinde sandbox'lanmış olduğundan, bare metal'e kıyasla çok daha az izin onayına ihtiyacınız olur. Container güven sınırıdır.

Autonomous Coding

Autonomes Programmieren

Otonom Kodlama

Hand off a task and walk away. The AI reads your codebase, writes code, runs tests, and iterates on errors. All without a single confirmation prompt.

Gib eine Aufgabe ab und geh. Die KI liest deine Codebasis, schreibt Code, führt Tests aus und iteriert bei Fehlern. Alles ohne eine einzige Bestätigungsaufforderung.

CI/CD Pipeline Development

CI/CD-Pipeline-Entwicklung

CI/CD Pipeline Geliştirme

Dank Docker-in-Docker können KI-Agenten Dockerfiles erstellen, Images bauen, Test-Services starten und komplette Deployment-Abläufe in der Sandbox validieren.

Docker-in-Docker sayesinde yapay zeka ajanları Dockerfile oluşturabilir, imaj derleyebilir, test servisleri başlatabilir ve tüm deployment akışlarını sandbox içinde doğrulayabilir.
